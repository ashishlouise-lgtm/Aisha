# Raj Laxmi Jewellers Telegram Bot

## Setup (Railway)

1. Add Environment Variables in Railway:
   - `BOT_TOKEN` = Your Telegram Bot Token
   - `GOLD_API_KEY` = Your GoldAPI.io Key

2. Deploy project on Railway.

3. Bot will automatically start and respond to:
   - `1` or "gold" → Shows live gold rate
   - `2` or "silver" → Shows live silver rate
   - `3` or "owner" → Shows owner contact
